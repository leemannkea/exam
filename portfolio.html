This first semester was comprised of four key learning areas: communication and presentation, design and visualisation, interaction development and business. My portfolio is organised thematically and aims to demonstrate knowledge learnt and skills aquired in these subject areas.

When we began this project HTML and CSS were not languages I spoke. The given style for my first responsive website was Swiss Style, renowned for its strong emphasis in objectivity, readability and cleanness.
